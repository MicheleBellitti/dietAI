# dietAI
AI Multi agent Meal Planner

## quick setup

```bash
cd backend
python3 db.py
```

```bash
cd ../frontend/
npm init
```
The follow the guided setup and your good to go.

## Usage

Once your'all set:

### Backend
```bash
cd backend
python3 app.py
```

### Frontend

```bash
cd ../frontend/
npm run dev
```
